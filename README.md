## Instalação :arrow_down:

### Pré-requisitos

| Tool                   | Versão Mínima | Download Link                               |
| ---------------------- | --------------- | ------------------------------------------- |
| **Node.js**            | v18.17.0+       | [Download](https://nodejs.org/en/download/) |
| **Git**                | Latest          | [Download](https://git-scm.com/downloads)   |
| **pnpm** (recomendado) | Latest          | [Install](https://pnpm.io/installation)     |

### Passos

1. **Clone o repositório:**
```bash
git clone https://github.com/Siegrain12/portfolio.git
cd portfolio
```

2. **Instale as dependências:**
```bash
pnpm install
# ou
npm install
```

3. **Configure variáveis de ambiente:**
```bash
cp .env.example .env.local
```

4. **Execute o servidor de desenvolvimento:**
```bash
pnpm dev
# ou
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.

## Uso :joystick:

### Customização de Dados

Todos os dados do portfólio são gerenciados através de arquivos em `utils/data/`:

- `personal-data.js` - Informações pessoais
- `experience.js` - Experiência profissional
- `projects-data.js` - Projetos
- `skills.js` - Habilidades técnicas
- `educations.js` - Formação acadêmica
- `contactsData.js` - Dados de contato

### Adicionar Foto de Perfil

Coloque sua foto em `public/profile.png` ou outro nome, e atualize em `personal-data.js`:

```javascript
profile: '/seu-arquivo.png'
```

### Configurar Formulário de Contato

Edite o arquivo `.env.local`:

```env
GMAIL_PASSKEY=sua_senha_app_gmail
EMAIL_ADDRESS=seu_email@gmail.com
```

## Deployment :rocket:

### Deploy para Vercel (Recomendado)

1. Push seu código para GitHub
2. Vá em [Vercel](https://vercel.com)
3. Importe seu repositório
4. Adicione variáveis de ambiente em Settings
5. Deploy automático!

### Deploy para Netlify

1. Conecte seu repositório GitHub
2. Build command: `npm run build`
3. Publish directory: `.next`
4. Configure as variáveis de ambiente
5. Deploy!
